# Representational State Transfer [[API|Application Programming Interface]]

Architectural style for designing networked applications. It utilizes standard [[HTTP]] methods to access and manage resources over the web.

1. The [[Client|client]] sends an [[HTTP]] request to an [[API]] endpoint
2. The server processes the request and generates the response
3. The [[Serveur|server]] send back the response (generally with the requested data)
